
## 环境说明

原项目使用`carla==0.9.12`，本电脑使用`carla==0.9.16`，新版本的相关文档说明见`Docs_Carla_UE4/`和`PythonAPI_Carla_UE4/`

## 环境创建
```bash
conda create -n carla_nav_312 python=3.12
conda activate carla_nav_312

pip install D:\Workspace\02_Playground\CARLA_Latest\PythonAPI\carla\dist\carla-0.9.16-cp312-cp312-win_amd64.whl

pip3 install torch torchvision torchtext torchaudio -f https://mirrors.aliyun.com/pytorch-wheels/cu128/

pip3 install Shapely networkx opencv-python matplotlib pygame
```
运行demo脚本
```bash
python data_collect.py --host 127.0.0.1 --port 2000 --map Town10HD --sync
```
